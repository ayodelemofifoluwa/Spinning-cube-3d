
# 3D Spinning Cube Animation

A simple 3D spinning cube animation built with HTML, CSS, and JavaScript.

## Demo
You can view a live demo of the animation [here](https://example.com/spinning-cube-3d).

## Features
* 3D spinning cube animation
* Pause animation on mouse enter, resume on mouse leave
* Customizable cube size, colors, and animation speed

## Installation
1. Clone the repository: `git clone https://github.com/your-username/spinning-cube-3d.git`
2. Open `index.html` in a web browser to view the animation

## Usage
* Customize the cube's size, colors, and animation speed by modifying the CSS styles
* Use the JavaScript code to add event listeners or modify the animation behavior
